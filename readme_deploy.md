# Como colocar sua Landing Page no ar (Gerar Link Público)

Atualmente, seus arquivos estão apenas no seu computador. Para ter um link "geral" (acessível por qualquer pessoa, no celular ou outro PC), você precisa hospedar o site.

Aqui está o método mais rápido e gratuito (Netlify Drop):

1.  **Acesse**: [https://app.netlify.com/drop](https://app.netlify.com/drop)
2.  **Arraste a pasta**: Pegue a pasta `Low - Ticket estrutura` onde estão seus arquivos (index.html, styles.css, imagens, etc) e arraste para dentro da área pontilhada no site.
3.  **Pronto**: O site vai gerar um link (ex: `site-bacana-123.netlify.app`).

## Outras Opções Profissionais

-   **Vercel**: Ótimo para projetos mais robustos. Instale o Vercel CLI e rode `vercel` na pasta.
-   **Hostinger/Hostgator**: Se você já tem hospedagem, acesse o Gerenciador de Arquivos (cPanel) e faça upload dos arquivos na pasta `public_html`.
